# SmartClicker

SmartClicker is a lightweight Windows auto-clicker built using Python and Tkinter.  
It automatically clicks when the mouse remains static and provides multiple speed and pattern options.

---

##  Features

- Auto-click on mouse inactivity
- Hotkey toggle (F6)
- Real-time CPS counter
- Multiple click speeds (Up to 30 CPS)
- Click patterns: Normal, Burst, Random
- Stealth mode (Human-like clicking)
- Persistent settings
- Clean and simple GUI
- Portable Windows EXE

## How to Use

1. Download `SmartClicker.exe`
2. Double-click to open
3. Choose speed and pattern
4. Press **F6** to Start / Stop
5. Keep mouse still to activate clicking
